JavaScript

an object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators

An operator is a symbol that tells the compiler to perform specific mathematical or logical functions.

Variables

A variable is a global variable if it's declared outside of a function (with or without "var").

Var. naming conventions

Naming and declaration rules for variables and functions. Rules for the use of white space, indentation, and comments.

Functions

Function arguments are the values received by the function when it is invoked.

Methods

JavaScript methods are the actions that can be performed on objects. A JavaScript method is a property containing a function definition.

Arguments

The arguments object is a local variable available within all (non-arrow) functions.

Parameters

Function parameters are the names listed in the function definition.

Return

The return statement stops the execution of a function and returns a value from that function.

Chaining Methods

Refers to repeatedly calling one method after another on an object, in one continuous line of code. Also known as Cascading.

Strings

A string is a series of characters that can be any text inside double or single quotes

Booleans

JavaScript booleans can have one of two values: true or false.
